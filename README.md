# Money Manager Full Stack

This repository contains both the Frontend and Backend of the Money Manager application.

## Structure
- **/frontend**: React.js application (Vite + TailwindCSS)
- **/backend**: Node.js/Express API with MongoDB Atlas

## Getting Started
To run the project locally, follow the instructions in the respective directories.
